# Roselyn Fine Dining
A Modern restaurant landing page built with a sophisticated black and gold design theme, crafted to reflect a premium dining experience.

## Features
- Elegant hero section with restaurant branding
- Interactive menu showcase
- Photo gallery
- Reservation/booking form
- Customer testimonials section
- Fully responsive design (mobile, tablet, desktop)

## Tech Stack
- **Frontend:** HTML, CSS
- **Design:** Custom luxury black and gold theme

## How to Run
1. Clone or download this repository
2. Open `index.html` in your browser
3. No additional setup required — pure HTML/CSS
